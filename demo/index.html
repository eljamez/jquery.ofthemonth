jquery.ofthemonth.js

A jQuery plugin that returns the js date object of a specific upcoming day.

This plugin makes it easy to get a "next specific date".

Returns in the form of a JavaScript Date Object

For example, let's say you want to always display the First Sunday of every month on your website.

All you have to do is call this plugin with the parameters you choose.

The next "First Sunday" is

The Paramaters Include

  • Which Week To Target ("which_week" in the JS)
  • Which Day To Target ("which_day" in the JS)

The Code

The JavaScript / jQuery

Just add "ofthemonth()" to your jQuery object.

// returns the default (last friday);
$(this).ofthemonth();

// optionally, you can set the defaults in your JS
$.fn.ofthemonth.defaults.which_day = "Sunday";
$.fn.ofthemonth.defaults.which_week = "first";

// or just send the params when calling the plugin (returns next Second Thursday)
$(this).ofthemonth({which_day:"Thursday",which_week:"second"});

// I like to set it to a variable
var my_otm_date = New Date( $(this).ofthemonth() );

There you have it. Enjoy.
